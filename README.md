# Cosilium-LLM

**Мульти-агентная аналитическая система высшего уровня**

Cosilium-LLM объединяет несколько LLM (ChatGPT, Claude, Gemini, DeepSeek) в единую систему для получения аналитических результатов, стабильно превосходящих уровень человеческих экспертов и инвестиционных комитетов.

## Возможности

- **Бизнес-стратегия** — анализ рынков, конкурентов, стратегических решений
- **Ресёрч** — глубокое исследование любых тем с перекрёстной верификацией
- **Инвестиции** — оценка проектов, risk-adjusted анализ, сценарное моделирование
- **Разработка** — архитектурные решения, code review, технический аудит
- **Методологический аудит** — проверка исследований, выявление когнитивных искажений

## Итеративный процесс

| Итерация | Название | Описание |
|----------|----------|----------|
| 1 | Независимый анализ | Каждый агент анализирует задачу самостоятельно |
| 2 | Adversarial mode | Агенты критикуют друг друга по 10 критериям качества |
| 3 | Синтез | Устранение слабостей, формализация финального результата |

## Формат результата

Каждый анализ включает:

1. **Аналитический отчёт** — структурированный документ с предпосылками и методологией
2. **Таблица выводов** — вероятности, риски, условия фальсификации
3. **Формализованный итог** — математические формулы с расшифровкой
4. **Рекомендации** — варианты решений с trade-offs
