# Cosilium-LLM

## Обзор проекта

Мульти-агентная аналитическая система высшего уровня, объединяющая несколько LLM для получения результатов, превосходящих уровень человеческих экспертов.

## Назначение

- Бизнес-стратегия
- Ресёрч
- Инвестиционный анализ
- Разработка
- Технический и методологический аудит

## Архитектура агентов

| Агент | Роль | Фокус |
|-------|------|-------|
| **ChatGPT** | Логический аналитик | Логика, противоречия, когнитивные искажения |
| **Claude** | Системный архитектор | Методология, интеграция, финальная редакция |
| **Gemini** | Генератор альтернатив | Гипотезы, сценарии, cross-domain аналогии |
| **DeepSeek** | Формальный аналитик | Данные, модели, математика, технический аудит |

## Итеративный процесс

```
Итерация 1: Независимый анализ
     ↓
Итерация 2: Взаимная критика (adversarial mode)
     ↓
Итерация 3: Синтез и формализация
```

## RAG-агенты

1. **Оптимальные промты** — самоэволюционирующие промты агентов
2. **Образы мышления** — паттерны величайших умов человечества
3. **Контекст задачи** — данные пользователя (высший приоритет)

## Ключевые принципы

```
Если можно посчитать — нужно посчитать.
Если нельзя посчитать — нужно объяснить почему.
Если нельзя фальсифицировать — вывод считается слабым.
```

## Формат результата

Каждый анализ должен содержать:

1. **Аналитический отчёт** — структурированный, с предпосылками и методологией
2. **Таблица выводов** — вероятности, риски, условия фальсификации
3. **Формализованный итог** — формулы с расшифровкой переменных
4. **Рекомендации** — варианты с trade-offs и количественным сравнением
