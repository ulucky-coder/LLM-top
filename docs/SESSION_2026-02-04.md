# Сессия 2026-02-04: Personal AI Assistant

## Идея продукта

**Цель:** Универсальный AI-ассистент в Telegram, объединяющий роли:
- Ассистент
- Ментор
- Стратег
- Аналитик
- Психолог

**Ключевое требование:** Полная долгосрочная память — бот помнит все взаимодействия, результаты, контекст.

## Ответы на вопросы (2026-02-05)

### Базовые параметры
- **Для кого:** Только для себя лично (personal AI)
- **Архитектура:** Мульти-агентная система
- **Память:** Полная — всё хранится
- **Взаимодействие:** Проактивное

### Детализация

| Параметр | Выбор |
|----------|-------|
| **Проактивность** | Напоминания о незавершённых решениях |
| **Режим диалога** | Чат (короткие) + Анализ (длинные) |
| **Психолог** | Работа с убеждениями, когнитивные искажения, adversarial mode |
| **Конфликт агентов** | Показать все точки зрения → я решаю. По команде: дебаты для консенсуса |
| **Приоритет данных** | 1) Бизнес, цели, метрики 2) Привычки, режим, здоровье |
| **Ключевое** | Глубокая и детальная аналитика |

## Архитектура (проект)

### Агенты

| Агент | Роль | LLM |
|-------|------|-----|
| **Strategist** | Бизнес-стратегия, цели, решения | Claude |
| **Analyst** | Данные, метрики, расчёты | DeepSeek |
| **Coach** | Убеждения, паттерны, challenge | Claude |
| **Tracker** | Незавершённые решения, напоминания | GPT-4 |
| **Synthesizer** | Финальный синтез, дебаты | Claude |

### Память

```
Memory Layer
├── Profile (facts)      — цели, ценности, контекст
├── Projects (active)    — текущие проекты, статусы, метрики
├── Decisions (pending)  — незавершённые решения (для проактивности)
├── Patterns (behavioral)— паттерны, привычки, здоровье
├── History (full)       — все диалоги (векторный поиск)
└── Insights (derived)   — выводы агентов, рекомендации
```

### Режимы работы

1. **Analysis** — глубокий анализ, все агенты (по умолчанию)
2. **Debate** — консенсус через adversarial дебаты (по команде)

### Ключевой принцип

> Это НЕ чат-бот для общения. Это персональный стратегический советник.
> Каждый ответ = конкретные рекомендации и решения для развития.
> Фокус: бизнес, работа, личная эффективность.
