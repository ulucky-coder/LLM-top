## API Integration

### Endpoints Used

```typescript
// Start analysis
POST /analyze/async
Body: { task, task_type, max_iterations, agents, settings }
Response: { task_id, estimated_duration }

// Get status (polling or SSE)
GET /tasks/{task_id}
Response: { status, progress, current_iteration, partial_results }

// Stream updates (SSE)
GET /analyze/stream?task_id={id}
Events: analysis_start, agent_complete, critique_start, critique_complete, synthesis_complete

// Export
POST /export
Body: { session_id, format: 'md' | 'pdf' | 'json' | 'csv' }
Response: { download_url }
```
